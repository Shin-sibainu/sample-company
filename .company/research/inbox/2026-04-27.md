# Claude Code リサーチ 2026-04-27

## 収集情報

### 1. Ultraplan — CLIからクラウドプランニングへ（Week 15: April 6–10）

- 概要: CLIからクラウド上にプランを下書きし、Webエディタでレビュー・コメントした後、リモート実行またはローカルへ引き戻せる「Ultraplan」が早期プレビュー入り。初回実行時にクラウド環境を自動作成する。同週に `Monitor` ツール（バックグラウンドイベントをストリーム）、`/loop` 自動ペース調整、`/team-onboarding`、`/autofix-pr` も追加。
- カテゴリ: 公式発表
- ソース: [What's new — Claude Code Docs](https://code.claude.com/docs/en/whats-new)
- 日付: 2026-04-06〜04-10
- 注目度: 高

### 2. Computer Use がCLIに研究プレビューで登場（Week 14: March 30 – April 3）

- 概要: ClaudeがターミナルからネイティブアプリのUI操作（クリック・検証）を行える「Computer Use」がCLIに実装。GUIでしか確認できないフィードバックループを閉じるユースケースに最適。`/powerup` インタラクティブレッスンや、MCP結果サイズ上限500K対応も同時追加。
- カテゴリ: 公式発表
- ソース: [What's new — Claude Code Docs](https://code.claude.com/docs/en/whats-new)
- 日付: 2026-03-30〜04-03
- 注目度: 高

### 3. Auto Mode — パーミッションプロンプト自動化（Week 13: March 23–27）

- 概要: 安全なアクションは自動承認、リスクの高いアクションはブロックするClassifierベースの「Auto mode」が研究プレビュー入り。`--dangerously-skip-permissions` と完全承認の中間地点として位置づけ。条件付きの `if` フックも追加。
- カテゴリ: 公式発表
- ソース: [What's new — Claude Code Docs](https://code.claude.com/docs/en/whats-new)
- 日付: 2026-03-23〜03-27
- 注目度: 高

### 4. Claude CodeのPro（$20）プランから一時削除騒動（2026-04-21〜04-23）

- 概要: 2026年4月21日、AnthropicがClaude CodeをProプランから削除し、Max 5x（$100/月）以上に変更する対応を実施。影響範囲は新規ユーザーの約2%とされたが、ランディングページやドキュメントが更新されたことで広く拡散。コミュニティの反発を受け、4月23日にAnthropicは誤りを認め変更を撤回。使用制限も4月23日にリセット。
- カテゴリ: 公式発表 / コミュニティ
- ソース: [Claude Code Removed from Pro Plan (pasqualepillitteri.it)](https://pasqualepillitteri.it/en/news/1211/claude-code-removed-pro-plan-anthropic-april-2026) / [wheresyoured.at](https://www.wheresyoured.at/news-anthropic-removes-pro-cc/) / [devtoolpicks.com](https://devtoolpicks.com/blog/anthropic-claude-code-pro-plan-removed-2026)
- 日付: 2026-04-21〜04-23
- 注目度: 高

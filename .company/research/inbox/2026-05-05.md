## トピック5: Claude Code 自律稼働強化 — VS Code拡張 GA・Checkpoints・Week 18 アップデート

- **日付**: 2026-05-04〜05-05（Week 18 / 自律強化発表）
- **カテゴリ**: 公式発表（機能追加）

### 概要

Anthropicが「Enabling Claude Code to work more autonomously」と題した公式記事を公開し、自律稼働を支える3つの柱を整理した: (1) Checkpoints（エージェントの各変更前に自動スナップショットを保存し、Esc×2押しや `/rewind` で即座に巻き戻し可能。30日間保持）、(2) VS Code拡張の正式GA（インラインdiff・`@`メンション・スラッシュコマンド・ワークスペース統合）、(3) Auto mode（安全な操作を承認なしで通過、リスク操作はブロックする分類器）。Week 18（5月4〜8日）のアップデートでは MCP `alwaysLoad` オプション追加・`/skills` への検索ボックス追加・PostToolUseフックが全ツールの出力を置換可能に・MCP起動時の自動3回リトライが実装された。

### なぜ注目されているか

Checkpointsは「Claude Codeに任せると怖い」という最大の心理的バリアを直接解消する機能。VS Code拡張のGAでターミナル外のユーザー層（IDE派）へのリーチが正式に確立された。Week 18のMCP強化はエコシステム成熟を示し、Claude Codeをハブとしたツール連携がより安定して機能するようになった。

### 引用・データ

- Checkpoints: 変更前に自動保存・30日保持・Esc×2または `/rewind` で巻き戻し
- VS Code拡張: 正式GA（インラインdiff・@メンション・スラッシュコマンド）
- Week 18 新機能: MCP `alwaysLoad`・`/skills` 検索ボックス・PostToolUse全ツール対応・MCP自動リトライ（最大3回）

### 1. Anthropic公式ポストモーテム — Claude Code品質低下の3つの原因

- **日付**: 2026-04-23
- **ソース**: [An update on recent Claude Code quality reports](https://www.anthropic.com/engineering/april-23-postmortem)
- **カテゴリ**: 公式発表
- **事実**: Anthropicは約1ヶ月間続いたClaude Codeの品質低下について公式ポストモーテムを公開。原因は3つ: (1) 3月4日にデフォルト推論努力を「high」から「medium」に変更、(2) 3月26日に投入したキャッシュ最適化バグが毎ターン思考履歴を消去、(3) 4月16日にシステムプロンプトでツール呼び出し間のテキストを25語以内に制限する指示を追加。全3件はv2.1.116（4月20日）で修正済み。4月23日に全サブスクライバーの使用制限をリセット。
- **考察**: モデルのネーフィング疑惑が実は「製品層のバグ3件の重複」だったという展開は衝撃的。ユーザーの体感とベンチマークの乖離、AIサービスの透明性問題を議論する好材料。Anthropicが自ら非を認めた誠実さも注目ポイント。
- **注目度**: 高
- **推奨投稿型**: 解説型

### 2. Claude CodeのProプラン削除実験 — 即日撤回の顛末

- **日付**: 2026-04-21〜22
- **ソース**: [Anthropic tests how devs react to yanking Claude Code from Pro plan](https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/) / [[UPDATED] News: Anthropic (Briefly) Removes Claude Code From Pro Plan](https://www.wheresyoured.at/news-anthropic-removes-pro-cc/)
- **カテゴリ**: 公式発表 / コミュニティ
- **事実**: 4月21日午後、AnthropicがProプラン（月額$20）の価格ページからClaude Codeのチェックマークを削除。成長責任者Amol Avasareが「新規Pro登録者の約2%に対するA/Bテスト」と説明したが、ランディングページやドキュメントが全体向けに更新されており混乱が発生。数時間内に元に戻し、「98%の人には関係のないテストだった」と謝罪。背景には、MaxプランがClaude Codeリリース前に設計されており、現在の使用量に対応しきれていないという課題がある。
- **考察**: 「A/Bテスト」という言葉でユーザーを実験台にする手法への反発と、有料AI依存のリスクが再燃。ローカルモデル（Llama、Mistral等）への移行議論が活発化した点も示唆的。
- **注目度**: 高
- **推奨投稿型**: 議論型

### 5. Claude Codeソースコードリーク — npmパッケージから512,000行が流出

- **日付**: 2026-03-31（影響は4月も継続）
- **ソース**: [Anthropic accidentally exposes Claude Code source code](https://www.theregister.com/2026/03/31/anthropic_claude_code_source_code/) / [Critical Vulnerability in Claude Code Emerges Days After Source Leak](https://www.securityweek.com/critical-vulnerability-in-claude-code-emerges-days-after-source-leak/) / [Claude Code leak used to push infostealer malware on GitHub](https://www.bleepingcomputer.com/news/security/claude-code-leak-used-to-push-infostealer-malware-on-github/)
- **カテゴリ**: 技術動向 / セキュリティ
- **事実**: 3月31日、npmパッケージ @anthropic-ai/claude-code に含まれた59.8MBのJavaScript sourcemapファイルから1,900ファイル・512,000行のTypeScriptソースコードが漏洩。研究者Chaofan ShouがXへ投稿して急拡散。Anthropicは「人的ミスによるリリースパッケージングの問題であり、セキュリティ侵害ではない」と声明。数日後にAdversa AIが重大な脆弱性を発見。また、GitHub上に偽のリークダウンロードを装ったVidar infostealerおよびGhostSocksマルウェアが拡散。
- **考察**: 「うっかりミス」が引き起こしたサプライチェーンリスクの典型例。オープンソースでないAIツールへの信頼性議論に新たな論点を追加。4月も余波（脆弱性・マルウェア）が続いているため直近性あり。
- **注目度**: 高
- **推奨投稿型**: 解説型

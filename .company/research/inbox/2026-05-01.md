### 1. Anthropic、Claude Code品質低下の原因を公式Postmortemで開示（4/23）

- **日付**: 2026-04-23〜2026-04-24
- **ソース**: [An update on recent Claude Code quality reports](https://www.anthropic.com/engineering/april-23-postmortem) / [The Register](https://www.theregister.com/2026/04/23/anthropic_says_it_has_fixed/) / [Fortune](https://fortune.com/2026/04/24/anthropic-engineering-missteps-claude-code-performance-decline-user-backlash/)
- **カテゴリ**: 公式発表
- **事実**: 3月26日〜4月16日にかけて独立した3つのエンジニアリング変更が重なり、Claude Codeの品質が約6週間にわたって低下した。①セッションのthinkingクリア処理にバグ（毎ターン発火）、②verbosity削減のシステムプロンプト追加がコーディング品質を悪化、③それらが重なり合って広範囲な劣化に見えた。4月20日（v2.1.116）までに全修正完了。Anthropicは4月23日に全サブスクライバーの使用制限をリセットした。
- **考察**: "ユーザーのせい" と示唆した初期対応からの謝罪・postmortem公開は透明性の観点で重要。AIツールの品質管理とコミュニケーションのあり方を問う事例として議論型投稿に最適。
- **注目度**: 高
- **推奨投稿型**: 議論型

### 3. Claude Opus 4.7がMax/Team Premiumのデフォルトに昇格、`xhigh` effortレベル追加（4/16）

- **日付**: 2026-04-16
- **ソース**: [Introducing Claude Opus 4.7](https://www.anthropic.com/news/claude-opus-4-7) / [GitHub Changelog](https://github.blog/changelog/2026-04-16-claude-opus-4-7-is-generally-available/)
- **カテゴリ**: 公式発表
- **事実**: Claude Opus 4.7がMaxおよびTeam Premiumプランのデフォルトモデルに昇格。新たに `xhigh` effortレベルを導入し、ほとんどのコーディング作業で推奨設定とされる。インタラクティブな `/effort` スライダーでリアルタイム調整が可能。CursorBenchでOpus 4.7は70%をクリア（Opus 4.6は58%）。自律動作でRust製テキスト読み上げエンジンをゼロから構築したことも実証済み。
- **考察**: "xhigh effort" という概念の導入はモデルの使い方を変える。ベンチマーク数値の改善とともに解説型投稿として訴求力が高い。
- **注目度**: 高
- **推奨投稿型**: 解説型

### 5. Routines機能でClaudeタスクをスケジュール/GitHub Eventから自動起動（4/13〜4/17）

- **ソース**: [SiliconANGLE: Anthropic's Claude Code gets automated 'routines' and a desktop makeover](https://siliconangle.com/2026/04/14/anthropics-claude-code-gets-automated-routines-desktop-makeover/)
- **カテゴリ**: 公式発表
- **事実**: Week 16（4/13〜4/17）で「Routines」機能がClaude Code on webに追加。Routineはプロンプト・リポジトリ・コネクタをパッケージ化し、スケジュール、GitHub Event、またはAPIコールをトリガーにAnthropicのクラウドインフラ上で自動実行される。ノートPCをシャットダウンしても継続動作する。あわせてCLIがネイティブバイナリに移行し起動速度が改善、`/usage` コマンドで使用状況の内訳確認が可能になった。
- **考察**: 「ノートPCを閉じてもエージェントが動き続ける」というメッセージは訴求力が強い。CI/CD的な自動化との比較も可能で、解説型・Tips型どちらにも応用できる。
- **注目度**: 高
- **推奨投稿型**: 速報型

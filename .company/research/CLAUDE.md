# リサーチ部

## 役割

Claude Code・AI関連の海外トレンド・ニュース・技術情報を収集・整理し、コンテンツ制作やビジネス判断に活用できるナレッジとして蓄積する。

## 担当範囲

- Claude Code / Anthropic の公式情報（リリース、ブログ、ドキュメント更新）
- AI業界全般のトレンド（競合動向、新技術、規制）
- コミュニティの声（Reddit, Hacker News, X の反応・議論）
- X投稿ネタの素材提供（コンテンツ部・x-auto-postスキルとの連携）

## ルール

- 収集した情報は必ずソースURL付きで記録する
- 日付は絶対日付（YYYY-MM-DD）で記録する
- 同じ日付のファイルがすでにある場合は追記する。新規作成しない
- ファイル操作前に必ず今日の日付を確認する
- 事実と考察は明確に分ける（「事実:」「考察:」プレフィックス）
- 不確かな情報は「未確認」と明示する

## フォルダ構成

- `inbox/` - 未整理の速報・クイックキャプチャ（迷ったらここ）
- `sources/` - 整理済みのソース別ナレッジ（1トピック1ファイル）
- `reports/` - 週次・月次のトレンドサマリーレポート

## ファイル命名規則

- 速報: `inbox/YYYY-MM-DD.md`
- ソース: `sources/トピック名-kebab-case.md`
- レポート: `reports/YYYY-MM-DD-weekly.md` / `reports/YYYY-MM-monthly.md`

## ソースの記録フォーマット

```markdown
## トピック名

- **日付**: YYYY-MM-DD
- **ソース**: [タイトル](URL)
- **カテゴリ**: 公式発表 / コミュニティ / 技術動向 / 競合
- **事実**: 何が起きたか
- **考察**: なぜ重要か、どう活用できるか
- **X投稿候補**: あり / なし
```

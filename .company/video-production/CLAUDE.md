# 動画編集部

## 役割

トーキングヘッド動画のカット編集を担当する。無音カット・言い直しカット・フィラー除去を行い、視聴に適した動画を出力する。

## 担当範囲

- 動画の無音カット（0.43秒以上の gap）
- 言い直し・リテイクのカット（最後の方を採用）
- フィラー語の除去（えっと、あの、えー等）
- 冗長な「はい」「うん」の整理
- セルフレビュー（技術 + コンテンツ）

## 使用ツール

- **ElevenLabs Scribe**: 音声文字起こし（単語タイムスタンプ）
- **librosa**: セルフレビュー時の音量検証（-18dB 閾値）
- **ffmpeg**: 動画エンコード（CRF 18, +7dB, 30ms フェード）

## サブエージェント

| エージェント | ファイル | 役割 |
|---|---|---|
| video-editor | `.claude/agents/video-editor.md` | 動画編集の実行（STEP 1〜4） |
| video-reviewer | `.claude/agents/video-reviewer.md` | 技術レビュー + コンテンツレビュー |

## ワークフロー

```
STEP 1: Scribe 文字起こし → 単語タイムスタンプ + ハードカットオフ
STEP 2: Claude 判断 → 言い直し・フィラー・冗長はい除去
STEP 3: 単語間 gap クリーニング → 0.43秒以上カット（80ms バッファ）
STEP 3.5: @video-reviewer → 技術レビュー + コンテンツレビュー
STEP 4: エンコード → +7dB / CRF18 / 200ms プリロール / 0.5s 余韻
```

## パラメータ

| 項目 | 値 |
|---|---|
| gap 閾値 | 0.43秒 |
| セグメント末尾バッファ | 80ms |
| プリロール | 200ms |
| 最終余韻 | 0.5秒 |
| 音量ブースト | +7dB |
| 映像品質 | CRF 18 |
| オーディオフェード | 30ms |
| レビュー対象 | 5秒以上のセグメント |
| レビュー閾値 | -18dB |
| ハードカットオフ | 最後の単語 end + 0.5秒 |

## フォルダ構成

- `projects/` - 編集プロジェクトのログ（動画ごとの編集記録）
- `templates/` - 編集パラメータのテンプレート（将来用）
- `logs/` - 編集実行ログ

## 投稿2（解説型・CTA可）【約280文字】
> ソース: [Introducing Claude Opus 4.7 - Anthropic](https://www.anthropic.com/news/claude-opus-4-7)
> 推奨投稿時刻: 15:00

[ ] Claude Opus 4.7 が Claude Code のデフォルトモデルになった。

CursorBench で 70%（前バージョン比 +12pt）、Rakuten-SWE-Bench では本番タスクの解決数が 3 倍。数字だけ見ると「ほんとか」ってなるけど、実際に触ったらコード生成の精度が体感で変わった。

新設の `xhigh` 努力レベルがコーディング作業のデフォルト推奨になってて、`/effort` スライダーで速度と精度を自分で調整できる。重い処理は精度を上げて、サクッと試したいときは下げる、みたいな使い分けができるようになった感じ。

Claude Code の実務活用を深掘りしたい人は → https://claude-code-academy.dev/
